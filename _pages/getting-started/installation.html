Fresh Installation

This instruction are for people that have experience manually installing Drupal and importing databases. If you do not feel comfortable performing these steps yourself, contact your server administrator for help. WARNING Do not import the provided SQL file into an existing Drupal database or you will lose all existing data.

Server/Hosting Requirements

  • Apache : >= 2.4.7 hosted on UNIX/Linux, OS X or windows
  • PHP: > 7.3
  • Composer:
  • Drush: > 9.0
  • Database: MySQL - 5.7.8 (MariaDB 10.3.7, Percona 5.7.8) or higher with an InnoDB-compatible primary storage engine

How to run a fresh Installation

Full Drupal installation zip was done just before launch, and may not contain all the other data added after the launch.

Step 1

A get the copy of ACERWC-Website.zip (This was shared with the ACERWC Team during the delivery of the project) and extract the archive somewhere on your computer.

Step 2

Copy of ACERWC-Website extracted folder to a location on your web server (Xampp or LAMP Stack) where you want your site to be accessed.

Step 3

Access your MySQL database and create a new DB (example acerwc_website)

image here

Import the SQL file ACERWC-Website/database.sql.zip to database for theme(e.g. acerwc_website). The tool PhpMyAdmin or SQLyog might be helpfull for managing your databases.

image here

Step 4

Install profile standard for drupal

Step 5

Login to your site with the username admin and the password admin and visit admin/people to change the admin username and password again, visit admin/config/system/site-information to change the site-information again

Go through the Drupal configuration found in admin/config options and update all the necessary information to reflect your website. Also be sure to visit /admin/config/media/file-system and update the Files Directories.

And visit admin/config/development/performance Clear all cache drupal.
